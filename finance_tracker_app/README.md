# Finance Tracker

A Flutter app with **charts**, **dark theme**, and **expense categories**. Generated with FlutterForge AI.

## Features

- **Dark / light theme** – Toggle in the app bar; dark theme by default
- **Expense categories** – Food, Transport, Utilities, Entertainment, Shopping, Health, Education, Other
- **Charts** – Pie chart for expense distribution by category
- **Period filters** – View expenses by day, week, month, or year
- **Local storage** – Data persisted with Hive

## Getting Started

1. Install [Flutter](https://flutter.dev/docs/get-started/install).
2. From this directory run:

   ```bash
   flutter pub get
   flutter run
   ```

## Dependencies

- `provider` – State management
- `hive` / `hive_flutter` – Local storage
- `fl_chart` – Pie and charts
- `intl` – Date/currency formatting
